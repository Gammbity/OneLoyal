## Route Model

- `/admin` - umumiy platforma admin paneli

- `/{company_name}/admin` - kompaniya admin paneli
- `/{company_name}/user` - kompaniya foydalanuvchi portali

Misol: `dusel` kompaniyasi uchun admin panel `http://localhost:5173/dusel/admin`, user portali esa `http://localhost:5173/dusel/user` bo‘ladi.
